# Python Mysql人脸考勤系统 python421

## 项目概述

本项目是一款基于Python语言和Flask框架的人脸考勤系统。通过集成Dlib库的特征点检测和深度学习模型进行人脸识别，使用Mysql数据库进行数据记录，适用于教学场景中需要进行考勤管理的联网设备。

## 技术栈

- **后端**: Python
- **Web框架**: Flask
- **人脸识别库**: Dlib
- **数据库**: Mysql

## 功能模块

- **人脸录入**
- **实时人脸识别**
- **上课签到**
- **考勤数据记录**
- **信息展示**

## 系统角色

- **学生**: 录入人脸信息，进行上课签到
- **系统管理员**: 管理考勤数据，监控系统运行状态

## 运行环境

- **操作系统**: 具有摄像头的联网设备
- **依赖**: Python环境，已安装Flask、Dlib、Mysql等相关库

## 部署步骤

1. 安装Python及相关依赖库
2. 配置Mysql数据库
3. 部署后端服务
4. 启动Web服务

## 目录结构

```
.
├── app                # Flask应用目录
│   ├── static         # 静态文件目录
│   └── templates      # 模板文件目录
├── database           # 数据库相关文件
├── models             # 数据模型目录
├── templates          # 前端页面模板
└── utils              # 工具函数目录
```

## 核心亮点

- **实时性**: 系统支持实时人脸识别和签到
- **易用性**: 学生在闲暇时即可录入人脸，上课时快速刷脸签到
- **精准度**: 采用Dlib库68特征点检测器，结合深度残差网络模型提高识别准确率
- **扩展性**: 基于模块化的设计，易于扩展和适应不同场景的需求
